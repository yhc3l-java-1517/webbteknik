Doctypes

Doctypes appear at the very top of the page. They tell the browser how the html on your page should be interpreted.

xhtml 1.1 doctype

xhtml 1.1 is a form of xml and the strictest doctype. Forcing xml rules on the document. The browser doesn't care though.

<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">

html5 doctype

The html5 doctype allows code to be written far more loosely.

<!DOCTYPE html>
<html>

Block elements

Block elements provide the main structure of the page. They consume the entire width available to them.

Headings

We represent headings with the <h1> element. The number represents the headings importance. 1 is the most important and 6 the least.

Divisions

The div element is used for parts of the document that can't be semantically described by other html block elements. It is usually used in combination with a class or id attribute.

<div class="article">
   Some very exciting news.
</div>

Paragraphs

The paragraph element encloses paragraphs.

<p>
   The cat sat on the mat.
</p>

Tables

A table in html is used to represent tabular data. It should consist of one or more rows containing one or more columns. Traditionally tables were used to structure entire pages. This is not a good practise. They should only be used where data can be classified as tabular.

Lists

There are three different types of list.

  • Ordered lists <ol>
  • Unordered lists <ul>
  • Definition lists <dl>

Ordered lists

Ordered lists are used to represent lists that have an order or weight

<ol> <li>Do first</li> <li>Do second</li> <li>Do third</li> </ol>

Unordered lists

Unordered lists are used to represent lists that have a no order or weight

<ul class="menu"> <li>Home</li> <li>Shop</li> <li>Contact</li> </ul>

Definition lists

Definition lists are used to represent an item term and it's data

<dl> <dt>Car</dt> <dd>4 wheel transportation device</dd> <dt>Motorbike</dt> <dd>2 wheel transportation device</dd> </dl>

Inline elements

An inline element does not break the flow of the page.

Span

The span element is the inline equivalent of a div. It can be used to markup elements that can not be described by any other element.

Example: I read the art of computer programming

Anchor (Link)

The anchor element is used for linking to other resources. The resource could be within your page or on another page

An external link to <a href="http://google.se">google</a>

An internal link to <a href="#block">the block section</a>

Strong

The strong element is used for lifting out text that is <strong>important</strong>

Emphasis

The emphasis element is also used for lifting out text in a sentance. It has slightly <em>lower status</em> than strong.

Images

Images can be included on your page with the <img> tag. They must have a src attribut pointing to the url of the image and an alternative text specified in an alt attribute.
